Projects and contributions

This is a list of my featured projects. 

Active projects

  • Zepl.vim — Lightweight and easy REPL integration for Vim. 
  • MdView — The flexible Markdown control for Xamarin.Forms. 
  • Ascribe — An alternative to EditorConfig. 

Completed projects

  • Photon.vim — An elegant dark Vim colour scheme with minimal syntax highlighting that cares about the little details. 
  • Thermoelectric generator — An attempt to create a cheap thermoelectric generator capable of charging a mobile phone. 
  • Org.vim — A very minimal Org mode and Outline mode plugin for Vim providing only syntax highlighting and folding. 

Contribute to

  • OmniSharp-Vim — I wrote the automatic installer for both MS Windows and Unix-like OSs.  I occasionally fix bugs, add features and help resolve issues.
