Projects and contributions

In my free time I work on various open-source and free-software projects. Occasionally I'll work on hardware projects.

Active projects

  • Ascribe - An EditorConfig alternative.
  • Photon.vim - An elegant, dark colour scheme with minimal syntax highlighting for Vim.
  • Gait - An experimental and intelligent UI framework project.
  • Org.vim - An Org-mode plugin for Vim providing only the absolute necessities.
  • 8-bit homebrew computer - I'm working on designing and building a simple 8-bit computer from scratch.

Completed projects

  • Thermoelectric Generator - Creating a thermoelectric generator capable of charging a mobile phone for as little money as possible.
  • Vivid.vim - An extremely minimal Vim plugin manager focused on providing users with maximum control over when their plugins are enabled. I no longer use this.

Archived/deprecated projects

  • txtdb - An experimental RDBMS and DB engine using CSV files to store tables.
  • ALIS - An Arch Linux installer. This was my first programming project, it is embarrassing how bad the code quality is.

Contributed to

  • OmniSharp-Vim - I wrote the automatic installer for both MS Windows and Unix-like OSs. I occasionally fix bugs and help resolve issues.
